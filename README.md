# UX-Giustizia

UX-Giustizia è un'applicazione offline open source per l'analisi investigativa di casi complessi, inclusi fenomeni di abuso sistematico.

L'app consente di organizzare e collegare elementi di un caso attraverso una struttura logica composta da:

* Fatti
* Prove
* Atti
* Circostanze
* Relazioni logiche tra elementi

## Obiettivo

Fornire uno strumento per la ricostruzione strutturata e verificabile delle dinamiche di un caso, facilitando l'analisi logica delle informazioni presenti in un fascicolo.

## Caratteristiche

* Funzionamento completamente **offline**
* Struttura basata su **relazioni tra elementi**
* Approccio orientato all'**analisi investigativa**
* **Open source**
