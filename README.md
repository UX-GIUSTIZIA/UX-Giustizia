# UX-Giustizia

UX-Giustizia è un'applicazione offline open source per l'analisi investigativa di casi complessi, inclusi fenomeni di abuso sistematico.

L'app permette di organizzare e collegare gli elementi di un caso attraverso una struttura logica basata su:

* Fatti
* Prove
* Atti
* Circostanze
* Relazioni tra elementi

## Obiettivo

Fornire uno strumento per la ricostruzione strutturata e verificabile delle dinamiche di un caso, facilitando l'analisi logica delle informazioni contenute in un fascicolo.

## Caratteristiche

* Funzionamento completamente **offline**
* Struttura basata su **relazioni logiche tra elementi**
* Approccio orientato all'**analisi investigativa**
* Progetto **open source**

## Modello logico (semplificato)

Il sistema organizza gli elementi di un caso tramite relazioni tra:

* Fatto
* Prova
* Atto
* Circostanza

Relazioni principali:

Fatto ← Prova

Fatto ← Atto

Prova ↔ Circostanza

Nota: il modello completo delle relazioni è più articolato e viene sviluppato all'interno del progetto.

## Struttura del sistema

UX-Giustizia organizza le informazioni di un caso attraverso quattro elementi fondamentali:

**Fatto**
Evento o situazione rilevante per il caso che deve essere analizzato o ricostruito.

**Prova**
Elemento informativo che conferma, contraddice o integra un fatto (documenti, testimonianze, registrazioni, ecc.).

**Atto**
Documento o attività formale collegata al caso (denuncia, querela, verbale, segnalazione, ecc.).

**Circostanza**
Elemento di contesto che può influenzare l'interpretazione di fatti o prove (tempo, luogo, condizioni, relazioni tra soggetti, ecc.).

Questi elementi possono essere collegati tra loro tramite relazioni logiche che permettono di analizzare la struttura complessiva di un caso.
